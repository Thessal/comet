## 1. Keywords

Keywords are reserved identifiers.

### Structural Keywords
- `Type`
- `Behavior`
- `Flow`
- `Fn`

### Type Keywords
- `Series`
- `DataFrame`
- `Matrix`
- `Vector`

### Logic/Control Flow
- `return`

## 2. Operators & Punctuation
- `(`, `)` : Parentheses (arguments list, constraint set definition, )
- `-` : Constraint set substraction operator
- `->` : Return type arrow for Behavior, Flow, Fn
- `:` : Constraints annotation in parameters
- `=` : Symbol definition within a Flow, or named arguments in calls of Flow, Fn
- `{`, `}` : Block delimiters for statements in Flow 
- `,` : Separator

## 3. Literals

- **Identifiers**: `[a-zA-Z_][a-zA-Z0-9_]*`
- **Strings**: Double-quoted `"string"`
- **Integers**: `123`
- **Floats**: `123.45`

## 4. Comments

- Line comments: `// ...`
- Block comments: `/* ... */`
